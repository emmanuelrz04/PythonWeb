# PythonWeb  

**Uma aplicação web full-stack construída com Python (Flask) no backend e HTML/CSS no frontend.**

A documentação detalhada explicando cada elemento do código — tanto frontend quanto backend — estará disponível na pasta **"HowWork"**.

---

## Nota Importante

A lógica da aplicação (em `app.py`) está atualmente documentada por meio de comentários detalhados dentro do próprio código. Isso foi feito para tornar o processo de desenvolvimento mais claro e fácil de acompanhar. Eventualmente, essa documentação será migrada para a pasta "HowWork".

---

## Principais Funcionalidades

- Sistema completo para criar, editar e publicar posts  
- Interface temática com identidade visual personalizada  
- Painel administrativo com autenticação e controle de acesso  
- Layout totalmente responsivo (funciona em desktop, tablet e mobile)  
- Implantado com sucesso em ambiente de produção  

---

## Tecnologias Utilizadas

- **Backend:** Python, Flask, Jinja2, SQLite  
- **Frontend:** HTML, CSS  
- **Deploy:** PythonAnywhere  
- **Controle de Versão:** Git, GitHub
